# Kinetix

[Kinetix](https://github.com/taller2-fiufit) es una aplicación para la gestión de actividades físicas.

## svc-xxx
Servicio para la gestión de xxx

## Instalación
Revisar el .env.example para ver las configuraciones necesarias.

Requiere de una base de datos Postgre.

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
